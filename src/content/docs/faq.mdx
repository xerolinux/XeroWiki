<div class="faq-card">
<h3>Will there be desktop spins other than KDE Plasma?</h3>
<p>No. Maintaining multiple desktop environments is not feasible as a solo project.</p>
</div>

<div class="faq-card">
<h3>Why is there a separate nVidia ISO?</h3>
<p>nVidia retired driver support for legacy 900/1000 series cards. The dedicated nVidia ISO targets RTX-series cards with modern open drivers, while the Standard ISO serves Intel and AMD systems. See <a href="/distro/#iso-variants"><strong>The Distro</strong></a> for details.</p>
</div>

<div class="faq-card">
<h3>What about Hyprland or other window managers?</h3>
<p>A Hyprland integration was attempted but discontinued. Archived code is available <a href="https://github.com/xerolinux/HyprXero-git"><strong>here</strong></a>.</p>
</div>

<div class="faq-card">
<h3>Why isn't Firefox or LibreWolf included by default?</h3>
<p>This was addressed in a dedicated post — <a href="https://xerolinux.xyz/updates/browser-situation/"><strong>read the explanation</strong></a>.</p>
</div>

<div class="faq-card">
<h3>Why doesn't the ISO include more software out of the box?</h3>
<p>To keep the distro lean and stable. More packages means more variables that can break things. The <a href="/xero-tool/"><strong>toolkit</strong></a> provides optional software categories so you install only what you need.</p>
</div>

<div class="faq-card">
<h3>Can I request a feature or package?</h3>
<p>Yes, but all requests are evaluated for stability impact and usefulness. Submit requests on <a href="https://fider.xerolinux.xyz"><strong>Fider</strong></a> or <a href="https://discord.xerolinux.xyz"><strong>Discord</strong></a>.</p>
</div>
